# ClassMate – Student Class Schedule Manager

## Project Description
ClassMate is a simple Python console-based application developed for a university Python course final exam. The project allows students to manage their daily class routine by adding, viewing, and deleting classes. It demonstrates the use of Python modules, functions, file handling and menu-driven programming.

## Features
- Add new class to schedule  
- View today's classes  
- View tomorrow's classes  
- Delete a class  
- Simple text-based menu system  
- Automatic data saving  
- No login system required  

## Technologies Used
- Python 3  
- Standard Python libraries (datetime, json, etc.)

## Project Structure
ClassMate/
- main.py – Main program file  
- addclass.py – Add class functionality  
- view.py – View today’s and tomorrow’s classes  
- delete.py – Delete class functionality  
- schedule.json – Data storage file  
- README.md – Project documentation  

## How to Run
1. Clone the repository:
   git clone https://github.com/your-username/ClassMate.git

2. Open the project folder:
   cd ClassMate

3. Run the project:
   python main.py

## Usage
After running the program, select an option from the menu:
1. Add new class to schedule  
2. View today's classes  
3. View tomorrow's classes  
4. Delete class  
5. Exit
